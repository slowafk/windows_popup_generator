# Windows Pop-Up Generator

This is a simple Python application that generates a Windows-style pop-up window. Users can enter a custom title and message, then click a button to display a message box.

## Features
- Enter a custom title for the pop-up.
- Enter a custom message to be displayed.
- Click a button to generate the pop-up.
- Simple and lightweight, built with `tkinter`.

## Requirements
- Python 3.x
- Tkinter (included with standard Python installation)

## Installation
1. Clone this repository:
   ```sh
   git clone https://github.com/slowafk/windows_popup_generator.git
   ```
2. Navigate to the project directory:
   ```sh
   cd windows_popup_generator
   ```
3. Run the script:
   ```sh
   python win95_popup.py
   ```

## Usage
1. Open the application.
2. Enter a title for the pop-up.
3. Type your message in the text box.
4. Click the "Show Pop-Up" button to display the Windows-style message box.
